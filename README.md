# SQL Analytics & Data Modeling

A project that shows the analytics-engineering core of a data platform — joins across tables, window functions for running and ranked metrics, CTEs for readable pipelines, and the layered modeling pattern (staging → marts) that turns messy source tables into trustworthy datasets. Built a small star-schema-style model over orders, customers, and products that an executive dashboard can query directly.

## Stack
- SQL
- Data Modeling
- Joins
- Window Functions
- Analytics Engineering
